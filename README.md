## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. 

### Prerequisites

You need :

* gcc compiler
* make


### Installing

To run the program, download sources, compile and execute.

Compile through 

```
make
```

And execute through

```
./c-bin/play
```
